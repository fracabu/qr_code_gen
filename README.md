<a name="english"></a>
## :gb: English

### Overview

A simple Python program to generate QR codes with a graphical interface. Easy to use for creating QR codes from text or URLs.

### Features

- **GUI Interface** - User-friendly graphical interface
- **Text to QR** - Convert any text to QR code
- **URL Support** - Generate QR codes from URLs
- **Save Images** - Export QR codes as images
- **Customization** - Size and format options

### Quick Start

```bash
git clone https://github.com/fracabu/qr_code_gen.git
cd qr_code_gen
pip install -r requirements.txt
python main.py
```
